# 游낺 Place the Rabbit Game

A simple Python console game where you place a rabbit 游낺 on a green 3x3 grid by entering the row and column numbers.

## How to Play

1. Run the script.
2. You will see a 3x3 grid filled with green circles 游릭.
3. When prompted, enter a two-digit number representing the row and column (e.g., `23` means row 2, column 3).
4. The rabbit 游낺 will be placed at the specified location.
